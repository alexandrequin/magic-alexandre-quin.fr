En React, la structure est essentielle, mais c’est la gestion d’état qui gère le flux de données et assure que chaque composant reçoit les bonnes informations au bon moment.

## Les outils natifs de React

- useState : Idéal pour les états locaux simples (ex. formulaire, toggle). Léger et efficace !

- useReducer : Parfait pour des états plus complexes (ex. panier e-commerce). Une approche plus contrôlée via des actions.

- Context API : Évite le prop drilling en partageant des états globaux comme l’authentification ou le thème.

Et pour les applications avancées ?  
Quand ces outils deviennent insuffisants, des bibliothèques comme Redux, MobX, Zustand prennent le relais pour mieux gérer les états complexes.

Bien gérer son state, c’est assurer une app scalable, fluide et maintenable !

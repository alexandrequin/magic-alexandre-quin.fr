Le format **WebP** est une alternative performante à JPEG et PNG, offrant une meilleure compression sans perte de qualité. Mais comment l’exploiter efficacement ? La réponse : **WebP en ligne de commande** !

## Pourquoi utiliser WebP ?

- Compression avancée : Des fichiers jusqu’à 30% plus légers que JPEG.

- Qualité préservée : Supporte la compression avec ou sans perte.

- Compatibilité large : Pris en charge par tous les navigateurs modernes.

## Comment convertir vos images en WebP ?

- Installez l’outil WebP : sudo apt install webp (Linux) ou brew install webp (Mac).

- Convertissez une image : cwebp image.jpg -o image.webp

- Ajustez la qualité : cwebp -q 80 image.jpg -o image.webp

- Optimisez un PNG avec transparence : cwebp -lossless image.png -o image.webp

**Résultat ?** Des images plus légères, un site plus rapide et une meilleure expérience utilisateur !
